# Template Next Js - Typescript

### Technology used:
    ✅ Redux
    ✅ Redux Toolkit
    ✅ Redux Thunk

### Install
Khởi chạy ở chế độ developer    
```bash
yarn install
```

### Run development
```bash
yarn dev
```

### Run production
```bash
yarn build & yarn start
```

### Create a slice (redux toolkit)
```bash
yarn generate slice
```

### Push git tự động
Chạy lệnh sau đây:
```bash
python push.py
```
